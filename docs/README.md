# Caddie Documentation Index

This `docs/` directory contains the submission documents for Caddie.


## What Is In This Folder

- `functional-spec/`: functional specification templates and final PDF
- `documentation/`: user manual, technical guide, and testing documentation.
- `proposal/`: proposal draft material
- `user-testing/`: user-testing materials and shared source packs
- `video-walk-through/`: video walkthrough materials and demo source docs
- `ethics.pdf`: project ethics document for user testing
- `Expo_Poster.pdf`: project expo poster
