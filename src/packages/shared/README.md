# Shared Contracts (TypeScript)

Shared TypeScript types for the Caddie web app. Import these through `@shared/index` so UI components and API client helpers use one contract surface.

## Files and purpose
- `README.md` - Overview of the shared TypeScript package.
- `index.ts` - Exported TypeScript types for hubs, sources, chat, moderation, analytics, guides, FAQs, reminders, and notifications.
- `package.json` - Package metadata for the shared module.
