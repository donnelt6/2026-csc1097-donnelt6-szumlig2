## Learning outcomes
By the end of the module, students should be able to:
- explain retrieval-augmented generation workflows
- compare vector retrieval, keyword retrieval, and reranking techniques
- evaluate information systems using grounded metrics
- discuss hallucination risks and citation-based trust signals
