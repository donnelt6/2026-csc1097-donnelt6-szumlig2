# Lab Access and Facilities Notice

## Main computing labs
- TL-102 is the primary first-year setup lab.
- TL-204 is reserved for CSC4042 lab teaching on Thursdays from 14:00 to 16:00.
- TL-310 is the postgraduate project lab and is not part of routine undergraduate access.

## Opening hours
- Standard lab opening hours are Monday to Friday from 08:00 to 21:00.
- Weekend access is available only to students with approved project access.

## Entry rules
- Students must carry their campus ID card at all times.
- Food is not permitted in any computing lab.
- Covered drinks are allowed except in TL-310.

## Software availability
- Python 3.11, Git, and PostgreSQL client tools are preinstalled in TL-102 and TL-204.
- GPU-enabled workstations are available only in TL-310.

## Printing and peripherals
- Loan laptops can be requested from the Library help point for up to 4 hours.
- HDMI adapters can be borrowed from the IT Help Desk.

## Incident reporting
- Faulty equipment should be reported to ithelp@northbridge.example with the room number and workstation label.
